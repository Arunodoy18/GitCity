# Simple City Example

The most basic GitCity usage — render a single user's city.

## Run

1. Start the backend: `cd ../../backend && npm run dev`
2. Open `index.html` in a browser (or use a local server)
3. Enter a GitHub username and click "Render City"

## What it demonstrates

- Initializing GitCity SDK
- Rendering a city for a single user
- Auto-rotating camera
- Building click events
