# Team Dashboard Example

Visualize an entire team's GitHub activity as a shared city.

## Run

1. Start the backend: `cd ../../backend && npm run dev`
2. Open `index.html` in a browser
3. Enter team member usernames (one per line)
4. Click "Build Team City"

## What it demonstrates

- Multi-user city rendering
- Sidebar with team stats
- Building click interaction
- Auto-rotating overview camera
